App desenvolvido para o teste da GlobalTec usando Ionic e Angular.

A ideia inicial era criar apenas a conexão com a api de forma simples e focar no layout, entanto achei interessante a possibilidade de baixar a imagem para galeria do celular.

Comecei fazendo em React pois já ter algum conhecimento, mas notei que ainda faltam alguns recursos na documentação então decidir mudar para angular, pois a documentação está mais completa.

Ainda precisa corrigir alguns bugs, como ele criar um arquivo vazio quando baixa a imagem, provavelmente algum erro ao criar o nome do arquivo.
